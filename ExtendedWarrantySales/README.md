# ExtendedWarrantySales

Generated .NET + Blazor application from Process Narration, BRD, and Figma designs.

## Architecture
- Backend: ASP.NET Core 8.0 Web API
- Frontend: Blazor WebAssembly
- Database: SQL Server

## Getting Started

1. Update connection string in appsettings.json
2. Run migrations: `dotnet ef database update`
3. Start API: `cd ExtendedWarrantySales.API && dotnet run`
4. Start Client: `cd ExtendedWarrantySales.Client && dotnet run`
